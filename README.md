Contador com Display 7 Segmentos usando PIC

Descrição:

Projeto de sistema embarcado utilizando microcontrolador PIC para controle de display de 7 segmentos, implementando lógica de contagem e exibição numérica.

Objetivo:

Desenvolver um contador digital capaz de exibir valores em um display de 7 segmentos, demonstrando controle de saídas digitais e lógica sequencial.

Tecnologias Utilizadas:

MikroC
Microcontrolador PIC
Display de 7 segmentos

Funcionamento:

Inicialização das portas digitais
Controle dos segmentos (a–g) do display
Implementação de lógica de contagem
Atualização contínua do valor exibido

Testes Realizados:

Verificação da exibição correta dos números
Teste de sequência de contagem
Validação das saídas digitais

Resultado:

O sistema apresentou funcionamento estável, exibindo corretamente a sequência numérica no display de 7 segmentos.
